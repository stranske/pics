Library 1a: Ask Clarifiers

Top Ideas (3)

  • “Ask me up to three clarifying questions before providing any solution.”
  • “Identify the ambiguous terms in my request and ask for specifics on each.”
  • “Summarize what you believe I am asking, then ask two questions to refine that understanding.”

Additional Prompts (7)

  • “What details are missing that would help you answer accurately?”
  • “List any assumptions you’re making and ask for confirmation on them.”
  • “Before proceeding, ask me to clarify the context around [key topic].”
  • “Which parts of my request could be interpreted in more than one way? Please specify.”
  • “Do you need sample input data or an example to understand my request better?”
  • “Can you identify five points of uncertainty in my prompt and ask me to clarify each?”
  • “Ask for constraints or boundaries (e.g., format, length, scope) that I should provide.”

Technical Prompts (5)

  • “List any missing variable definitions or data structures you need to run this Python function.”
  • “What Excel column headers or sample rows are required to build the correct formula?”
  • “Show me which parts of my SQL query are ambiguous and ask for table schema or field names.”
  • “Identify any undefined functions or modules in this code snippet and ask me for details.”
  • “What unit tests or example outputs do you need to verify this algorithm?”

Library 1b: Restate Request

Top Ideas (3)

  • “Paraphrase what I just asked, and confirm if you have captured my intent correctly.”

  • “List the key tasks you believe you need to perform to solve my request.”
  • “Restate the desired outcome, then ask me if there are any edge cases to consider.”
  • “Repeat my query in your own words, then ask if you should interpret it as [possible interpretation].”
  • “Rewrite my prompt as a bullet-point checklist of requirements.”
